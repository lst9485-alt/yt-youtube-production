# 30대 평균 자산 현실, 꼭 해야 할 5가지 — 자료3 (내 영상/상담)

## 관련 기존 영상 목록 + 재활용 포인트

### [39번] 가난해지는 5가지 공통점
- 저축률 50% 미만 = 가난해질 확률 매우 높음
- "안사면 100% 할인" / 안뜯은 택배 = 소비중독 신호
- 소비빨대를 전부 뽑고 → 불편한 것만 작은거로 다시 꽂기
- "소비가 아니라, 극복이 진짜 경험"

### [36번] 친구와의 격차
- "반지하살면서 할부로 샤넬백 메는게 창피한거지, 내집에서 에코백은 부끄러운게 아님"
- 인스타 자랑하는 사람들 = 막상 보면 집도 없고 통장도 없는 경우 多
- 남들 시선이 무서울까, 나중에 내 자식 시선이 무서울까
- 자산격차 → 관계격차 → 대화 단절 → 자연스레 멀어짐
- "돈 벌더니 변했네?" → 생각이 변했기 때문에 돈을 번 것

### [60번] 월급날 루틴
- 통장 2개(고정비+변동비), 선저축 후지출
- 신용카드 가위로 자르기 — 대니얼 카너먼(노벨경제학상) '전망이론': 잃는 고통 = 얻는 기쁨의 2~3배
- 카톡 친구 1명당 연 20만원 지출 (60명 = 월 100만)
- 장보기 주 1회, 무지출데이 주 2회

### [14번] 1억부터 돈이 미친듯이 불어남
- 1억 = 돈의 태도를 바꾸는 전환점
- 1억 10% 수익 = 월급 4개월치 (월 250만 기준)
- 1억→10억: 6년 (채널 운영자 본인 사례)
- 초기 1,000만원 → 10억 약 8년, 10억 → 50억 약 2.5년 (복리 가속 구조)
- 소득 + 투자 + 시스템 = 3가지가 동시에 일해야

### [7번] 흙수저도 10억 가는 5단계
- 서울 아파트 중위값 10억, 최근 10년간 연 13% 상승 — KB부동산 시세 기준

| 단계 | 목표 | 핵심 |
|------|------|------|
| 1 | ~천만원 | 흑자구조, 빚청산 |
| 2 | ~5천만 | 투자공부, 몸값 올리기 |
| 3 | ~1억 | 4억대 아파트 매수 |
| 4 | ~3억 | 갈아타기, 근로+투자소득 |
| 5 | ~10억 | 투자수익 > 근로소득 |

### [17번] 0원→10억 로드맵
- 1천만까지 예적금만 / 5천만까지 ETF / 1억부터 부동산
- "5년 안에 1억 못 모으면 평생 10억 근처도 못 간다"
- 본인: 6년간 20억 벌어 10억 모음

### [67번] 재테크 기초
- 시중 유동성 매년 평균 7%씩 증가 → 10년마다 물가 2배 — 한국은행 통화량(M2) 통계
- 좋은 빚(집대출) vs 나쁜 빚(명품할부)
- 집값 5억 + 대출 3억 → 10년 후 집값 10억, 대출 실질가치 반토막

### [59번] 열심히 일할수록 가난해지는 이유
- 덧셈 사고(월급) vs 곱셈 사고(복리) = 핵심 차이
- "인형 눈깔 붙이기 부업을 신의 손놀림으로 해도 부자 못됨. 공장을 차려야"
- 벌고 × 모으고 × 불리기 → 하나라도 0이면 결과 0

### [32번] 집 산 친구 2억 벌었대요
- 전세는 비용, 집은 자산
- 월 200만원 x 4년 저축 = 1억 vs 집값 2~3억 상승 → 따라잡기 불가
- "무주택은 후진(R), 1주택이 중립(N)"
- 마포 신혼부부 2쌍: 전세 vs 매수 → 2년 만에 자산 차이 2억 (채널 자체 사례)

### [65번] 집 안사고 버티면
- 월세 = "가난을 구독하는 무서운 구독 서비스"
- 전국 월세 비중 66.0% (역대 최고, 2024.8 기준) — 한국부동산원

### [63번] 흙수저가 겪게 될 현실
- 3년 안에 1억 모으기 / 내 집 무조건 마련 / 수입 늘리기 / 환경 바꾸기
- 김씨 vs 박씨 실화: 신축전세+외제차 → 경기외곽으로 밀려남 vs 구축영끌 → 5년 뒤 10억

### [43번] 5년 뒤에도 가난한 사람들
- NH투자증권: 부자의 80% 이상이 자가에 거주
- "수익만 복리로 불어가는 게 아니라, 가난도 복리로 쌓인다"

### [5번] 부자가 될 수 있는 5가지 비밀
- 자주 만나는 5명 평균 = 나, 내 주위 가장 성공한 사람 = 천장
- "태어날때 가난한 건 괜찮지만 죽기전 가난은 내 책임" (빌게이츠)

### [42번] 부자 되기 직전 3가지 현상
- 정체성 변화("나는 부자다") → 행동 변화 → 관계 변화
- "4~50대는 2~30대의 성적표"

### [13번] 코인투자로 2억 날리고
- 2억 2천 투자 → 2천만원 회수 (90%+ 손실)
- 50% 손실 복구 = 100% 수익 필요
- Risk(감당가능) vs Danger(죽을수도) 구분 필수

### [47번] 벼락거지 되는 확실한 방법 (풍자)
- 28세 "괜찮음" → 31세 "친구들 진급" → 35세 "초조" → 40세 "알바도 안불러줌"
