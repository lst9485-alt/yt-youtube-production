## 기술 결정
### 1_무음제거.py
- Phase 1: ffmpeg silencedetect (기본: -35dB, 0.4s) → 러프 컷 (고정 pad 0.5s)
- Phase 2: faster-whisper base 모델, cpu/int8, 구간별 전사
- Phase 3: segment 모드(2초 gap) / tight 모드(--gap 기준 단어), 비대칭 패딩 지원
- Phase 4: re-encode (libx264 fast crf18 / aac 192k) → concat demuxer copy
- SRT: 컷 후 타임라인 기준 재계산, --filter-fillers로 필러 단어 제외 가능
- --no-refine: Silero VAD 경계 미세조정 비활성화용 (기본: 활성, ±0.5s 윈도우, threshold=0.35)
- 기본 padding: 0.5s (--pad-start로 시작, --pad-end로 끝 개별 조정 가능)
- 이전 기본값 복원: --silence-thresh -30 --min-silence 0.5 --padding 0.4

### 3_컷편집.py
- SRT 파싱: 빈 줄 기준 블록 분리, index 필드 보존 (리테이크 감지용)
- 구간 병합: padding(0.4s) 추가 후 gap(2.0s) 이하면 병합, 비대칭 패딩 지원
- 리테이크 경계: SRT index 불연속 시 gap 이하여도 병합 차단 (--no-retake-detect로 비활성)
- 추출: re-encode (libx264 fast crf18 / aac 192k) → concat copy
- SRT 재계산: 각 segment의 누적 오프셋으로 타임코드 재매핑

### 4_효과음.py
- faster-whisper base 모델 (한국어, word_timestamps=True)
- $$ 먼저 매칭 후 $ 처리 (플레이스홀더 방식)
- 마커 직전 단어 end + offset(-0.05s) = 삽입 시간
- 무음 트랙 + 효과음 overlay → WAV
