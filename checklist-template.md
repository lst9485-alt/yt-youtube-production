## 주제 선정 ← yt01-topic

스킬 진행:
- [ ] STEP 1: 키워드 검색량 검증

- [ ] STEP 3: 주제 후보 생성
- [ ] STEP 4: 프로덕션 폴더 생성
- [ ] STEP 5: 체크리스트 점검

품질 점검:
- "이 영상은 결국 무슨 말을 하려는 걸까?"
- 나는 [A]에게 [B]를 줄 것이다
    - 나는 [A]가 겪는 [B]라는 문제를 해결 (교훈, 새로운 사실, 인사이트 등)
    - 나는 [A]에게 [B]라는 감정을 느끼게 (공감, 감동, 위로, 궁금증 해소, 짜릿함 등)
    - [ ] A(어떤 사람) :
    - [ ] B(어떤 문제, 감정) :
- 대중성이 있는가?
    - [ ] A 코어 시청자
    - [ ] B 관심 시청자
    - [ ] C 관심x 시청자
    - 내 팬이 아니라, 나를 모르는 유입 관점에서 만들었는가? (아웃라이어)
- 노출이 잘 되는 주제인가?
    - [ ] 검색어 트렌드
    - [ ] 뷰트랩 성장속도
    - [ ] 유튜브 검색 등

## 썸네일 & 제목 ← yt02-thumbnail

스킬 진행:
- [ ] 1단계: 제목 + 썸네일 문구 3세트
- [ ] 2단계: 비주얼 가이드

품질 점검:
- 썸네일 : 정말 '욕구'를 공략하고 있는가?
    - [ ] 기대/증거/의문/공감?
    - [ ] 글자/그림 비중?
    - [ ] 3개 만들었는가? (간단하게)
    - [ ] 디밸롭 여부 (핫비디오, 성공구조, 제미나이 등)
- 제목 : 내 제목은 목적에 부합하는가?
    - [ ] 시청자 머리에 [물음표] 띄웠는가?
        - [ ] 중요정보 누락 5W1H (누가, 언제, 어디서, 무엇을, 어떻게, 왜)
    - [ ] 제목 ≠ 내용인가?
    - SEO 확인 (유튜브 검색)
    - 성공 문장구조, 트렌드, 상위어 등을 활용했는가?
- [ ] 시청자가 원하는 혜택인가? (피해보다는 혜택)
    - 부정적 표현 추가, 극단적 표현 추가 (필수x)
